# AI Engineering Lab 🚀

A hands-on repository documenting my journey into AI Engineering through practical projects.

This repository contains experiments, prototypes, and production-inspired applications covering Large Language Models (LLMs), AI Agents, Retrieval-Augmented Generation (RAG), Model Context Protocol (MCP), prompt engineering, and backend integrations using Python and Java.

## 🎯 Goals

- Learn AI Engineering by building real-world projects
- Explore modern LLM application architectures
- Build production-ready AI agents
- Practice prompt engineering and evaluation
- Integrate AI with backend systems and APIs
- Document lessons learned throughout the journey
